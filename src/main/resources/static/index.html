FinanceMaster API

Frontend Zugang:

  • Live (Production): https://frontend-7vbb.onrender.com
  • Lokal (Dev): http://localhost:5173/

1. Authentication & Session

  • POST /auth/register
    Neuen Benutzer registrieren.
  • POST /auth/login
    Anmelden (Setzt HttpOnly Session Cookie).
  • GET /auth/me
    Prüft, ob eine Session aktiv ist und gibt den User zurück.
  • POST /auth/logout
    Beendet die Session serverseitig.

2. Categories

  • GET /categories
    Lädt alle Kategorien des angemeldeten Benutzers.
  • POST /categories
    Erstellt eine neue Kategorie.
  • DELETE /categories/{id}
    Löscht eine Kategorie (nur wenn keine Transaktionen verknüpft sind).

3. Transactions (Filter)

  • GET /transactions
    Lädt die Transaktionsliste. Unterstützt Filterung.

    Query Parameter (Optional):

    • ?categoryId=1 (Filtert nach einer Kategorie-ID)
    • ?from=2025-01-01 (Startdatum, Format YYYY-MM-DD)
    • ?to=2025-01-31 (Enddatum, Format YYYY-MM-DD)
  • POST /transactions
    Erstellt eine neue Einnahme oder Ausgabe.
  • DELETE /transactions/{id}
    Löscht eine Transaktion anhand ihrer ID.

4. Summary & Dashboard

  • GET /transactions/summary/balance
    Liefert Summen für Einnahmen, Ausgaben und die Bilanz.

    Unterstützt dieselben Filter wie die Liste (categoryId, from, to).
