FinanceMaster API

REST API für Finanzverwaltung

Authentication

  • POST /auth/register - Benutzer registrieren
  • POST /auth/login - Benutzer anmelden
  • GET /auth/me - Aktuellen Benutzer abrufen (Auth erforderlich)
  • POST /auth/logout - Benutzer abmelden (Auth erforderlich)

Categories

  • GET /categories - Alle Kategorien abrufen (Auth erforderlich)
  • POST /categories - Kategorie erstellen (Auth erforderlich)
  • DELETE /categories/{id} - Kategorie löschen (Auth erforderlich)
  • DELETE /categories - Alle Kategorien löschen (Auth erforderlich)

Transactions

  • GET /transactions - Alle Transaktionen abrufen (Auth erforderlich)
  • POST /transactions - Transaktion erstellen (Auth erforderlich)
  • DELETE /transactions/{id} - Transaktion löschen (Auth erforderlich)
  • DELETE /transactions - Alle Transaktionen löschen (Auth erforderlich)

Summary

  • GET /summary/balance - Balance-Übersicht mit optional Filterung nach Kategorie, Datum (Auth erforderlich)
  • GET /summary/by-category - Übersicht gruppiert nach Kategorien (Auth erforderlich)
  • GET /summary/by-date - Übersicht gruppiert nach Datum (Auth erforderlich)
